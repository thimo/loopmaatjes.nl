Changelog: Buddy

If you are upgrading from a version earlier than 1.4 please read the following:
  • The "Slide URL", "Link Type", and "Caption Position" slider settings will need to be reapplied for individual slides.
If you are upgrading from a version earlier than 1.1.1 please read the following:
  • The Site Wide Activity widget has now been replaced with an in-built Activity Stream shortcode. Edit your homepage and find: [sidebar name="gp-homepage" text="Go to Appearance -> Widgets and drag and drop the Site Wide Activity widget into the Homepage widget area on the right. If you have not installed this widget yet, see http://ghostpool.com/help/help.html#swa-plugin"]
  • Replace with [activity header="Latest Activity" per_page="5" comments="threaded" order="desc" pagination="true"]
  • For more information on the Activity Shortcode click here.
